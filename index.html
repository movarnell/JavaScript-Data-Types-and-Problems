Javascript Data Types

JavaScript is a dynamically typed language, which means that variables can hold values of different types. Understanding the different data types in JavaScript is essential for writing effective and bug-free code.

Primitive Data Types

JavaScript has six primitive data types:

  • Number: Represents numeric values, such as 42 or 3.14.
  • String: Represents textual data, enclosed in single or double quotes.
  • Boolean: Represents either true or false.
  • Null: Represents the intentional absence of any object value.
  • Undefined: Represents an uninitialized variable or missing property.
  • Symbol: Represents a unique identifier.

Composite Data Types

JavaScript also has composite data types:

  • Object: Represents a collection of key-value pairs.
  • Array: Represents an ordered list of values.

Example Problems

Here are some example problems to help you understand JavaScript data types:

  1. Create a variable called age and assign it the value 25.
  2. Create a variable called name and assign it your name as a string.
  3. Create a variable called isStudent and assign it the value true.
  4. Create a variable called person and assign it an object with properties like name, age, and isStudent.
  5. Create an array called fruits and add some fruit names to it.
